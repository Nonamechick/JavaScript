JavaScript Number Methods

The toString() method converts a number to a string.

The toExponential() method returns a string, with the number rounded and written using exponential nonation

An optinal parameter defines the number of digits behind the decimal point

To toFixed() method rounds a number to a given number of digits

For working with money, toFixed(2) is perfect

The toPresion() method returns a string, with a number written with a specified lenght

The valueOf() method returns a number as a number
